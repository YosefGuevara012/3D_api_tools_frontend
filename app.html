Welcome to 3DTracking Toolkit

This toolkit provides a set of utilities to manage and process data within the 3DTracking platform.

Tool Description
Unit Creation tool Automatically creates units and associated trackers from imported data.
Update Attributes tool Bulk updates vehicle attributes across multiple units.
From CSV to KML Converts platform .csv files into .kml format.

Use the left menu to navigate between available functionalities.

Unit Creation

This tool implements a tracker and unit creation within the 3Dtracking platform. Please upload an excel file with the required unit information to proceed with the creation process.

Drag 'n' drop the file here, or click to select the file.

Update Attributes

This tool allows you to update attributes of existing units. Select the attributes you wish to modify and apply the changes. Please upload an excel file with the required unit information to proceed with the creation process.

Drag 'n' drop the file here, or click to select the file.

From .csv to .kml converter

This tool allows you to convert the platform's location list report from csvexport (exported as .csv) into the standard (Keyhole Markup Language) .kml format. Please upload the .csv file to proceed with the conversion.

Drag 'n' drop the file here, or click to select the file.
